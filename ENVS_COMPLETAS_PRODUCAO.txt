============================================
COMO OBTER OS VALORES [OBTER]
============================================

SUPABASE:
1. Acesse: https://supabase.com/dashboard
2. Selecione seu projeto
3. Settings → API:
   - Project URL → NEXT_PUBLIC_SUPABASE_URL
   - anon public → NEXT_PUBLIC_SUPABASE_ANON_KEY
   - Service Role (clique no olho) → SUPABASE_SERVICE_ROLE_KEY
4. Settings → Database:
   - Connection string → URI → DATABASE_URL

ASAAS:
1. Acesse: https://www.asaas.com
2. Faça login
3. Configurações → Integrações → API
4. Copie a chave de produção (aact_prod_...)

OPENAI:
1. Acesse: https://platform.openai.com/api-keys
2. Clique em "Create new secret key"
3. Copie a chave (ela só aparece uma vez!)

RESEND (EMAIL):
1. Acesse: https://resend.com
2. Crie uma conta gratuita
3. API Keys → Create API Key
4. Copie a chave (re_...)

GOOGLE ANALYTICS 4:
1. Acesse: https://analytics.google.com
2. Admin → Data Streams
3. Clique no stream do seu site
4. Copie o Measurement ID (G-...)

NEXTAUTH_SECRET:
Execute no terminal:
openssl rand -base64 32
Copie o resultado

GO HIGH LEVEL (GHL) - OPCIONAL:
1. Acesse: https://app.gohighlevel.com
2. Settings → Integrations → API → Copiar token
3. Settings → Locations → Copiar Location ID
4. Pipelines → Selecionar pipeline → Copiar Pipeline ID
5. Pipelines → Estágios → Copiar IDs dos estágios

STRIPE - OPCIONAL:
1. Acesse: https://dashboard.stripe.com
2. Developers → API keys → Copiar Secret key
3. Developers → Webhooks → Criar webhook → Copiar Signing secret
