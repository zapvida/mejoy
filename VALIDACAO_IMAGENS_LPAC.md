## 📝 NOTAS TÉCNICAS

### Como funciona:
- **object-fit: cover** - A imagem preenche todo o container mantendo proporção
- **object-position** - Define qual parte da imagem fica visível
- **overflow-hidden** - Esconde partes que saem do container
- **aspect-square/aspect-circle** - Mantém formato perfeito

### Performance:
- Imagens otimizadas pelo Next.js Image
