# ⚡ ALINHAR CONTAS AGORA

Execute este comando para alinhar tudo com `aistoteleapp-art`:

\`\`\`bash
pnpm force:vercel-account
\`\`\`

O script vai guiar você através do processo de logout/login com a conta correta.

**Depois de executar, valide:**
\`\`\`bash
pnpm validate:accounts
\`\`\`

Tudo deve estar ✅ quando terminar!
